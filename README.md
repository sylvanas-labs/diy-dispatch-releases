# DIY Dispatch — Releases

This repository hosts public release artifacts for **DIY Dispatch**, a multi-provider phone-to-PC task dispatcher supporting Claude, Codex, Aider, and OpenAI.

> Source code: [sylvanas-labs/diy-dispatch](https://github.com/sylvanas-labs/diy-dispatch) *(private)*

## Downloads

Go to the [Releases](../../releases) tab to download the latest version.

| Platform | File | Notes |
|----------|------|-------|
| Android | `DIY-Dispatch-Android-<version>-beta.apk` | Sideload — see install instructions below |
| Windows | `DIY-Dispatch-Setup-<version>.exe` | NSIS installer |

## Release Tracks

| Track | Tag | Description |
|-------|-----|-------------|
| **Stable** | `stable-latest` | Built from `main`. Recommended for most users. |
| **Beta** | `beta-latest` | Built from `develop`. May contain incomplete features or bugs. |

## Installing on Android

1. On your Android device, enable **Install unknown apps** for your browser or file manager (Settings → Apps → Special app access).
2. Download the latest `.apk` from the [Releases](../../releases) tab.
3. Open the downloaded file and tap **Install**.

## Installing on Windows

Run `DIY-Dispatch-Setup-<version>.exe` and follow the installer prompts.
